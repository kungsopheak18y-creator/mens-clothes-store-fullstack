# Men's Clothing Backend – Full Codebase Summary

## Overview

This backend is a secure e-commerce REST API built with:

* Node.js
* Express.js
* Prisma ORM
* PostgreSQL
* JWT Authentication
* Cloudinary image upload
* Bakong KHQR payment integration

The project is designed for a men's fashion e-commerce website with:

* User authentication
* Product management
* Categories & brands
* Orders
* KHQR payments
* Address management
* Admin controls

## Files

```txt

```

---

## Registration Flow

### Route

```http
POST /api/auth/register
```

### Process

1. Validate input
2. Check duplicate email/phone
3. Hash password with bcrypt
4. Create user
5. Generate JWT token
6. Return token + user info

---

## Login Flow

### Route

```http
POST /api/auth/login
```

### Features

Supports login using:

* Email
* Phone number

### Process

1. Find user
2. Compare password
3. Generate JWT
4. Return user data

# 9. Payment System (Bakong KHQR)

This backend integrates Cambodia's Bakong KHQR system.

Files:

```txt
controllers/payment.controller.js
services/bakong.service.js
```

---

## Payment Flow

### Step 1: Create Order

User creates an order.

Order status:

```txt
PENDING
```

---

### Step 2: Generate KHQR

Endpoint:

```http
POST /api/payment/initiate/:orderId
```

System:

* Generates KHQR string
* Creates MD5 token
* Stores expiration time

Returns:

```json
{
  "qrString": "...",
  "amount": 120,
  "expiresAt": "..."
}
```

---

### Step 3: Check Payment Status

Endpoint:

```http
GET /api/payment/status/:orderId
```

The backend:

* Calls NBC API
* Verifies MD5 transaction
* Updates order to PAID

---

### Step 4: Cancel Payment

Endpoint:

```http
DELETE /api/payment/cancel/:orderId
```

Logic:

* Restore stock
* Delete order items
* Delete order

---

## KHQR Features

### QR Expiration

QR expires after:

```txt
10 minutes
```

---

## Payment Status Handling

Possible states:

| Status  | Meaning            |
| ------- | ------------------ |
| PENDING | Waiting payment    |
| PAID    | Payment successful |
| EXPIRED | QR expired         |
